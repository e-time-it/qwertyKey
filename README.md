## Getting Started

Clone the repo and make it yours:

```bash
git clone https://github.com/e-time-it/qwertyKey
```

Install dependencies:

```bash
npm install
```

## Running Locally

```bash
npm start
```

## Running in Production

```bash
npm start
```
